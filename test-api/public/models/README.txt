여기에 올드 트래포드 3D 모델 파일을 넣으세요.

파일명(정확히): old-trafford.glb

받는 곳 (무료, CC-BY 출처표기 조건):
https://sketchfab.com/3d-models/old-trafford-manchester-united-037303d1456e4a7dbab111aab32494de

단계:
1) 위 링크 접속 → 무료 Sketchfab 계정 로그인
2) "Download 3D Model" → glTF (.glb) 선택
3) 받은 .glb 파일을 이 폴더에 "old-trafford.glb" 이름으로 저장
4) 브라우저 새로고침 → 경기장이 라인업과 함께 뜸

파일을 넣은 뒤 선수가 잔디 위에 안 맞으면,
src/Stadium3D.jsx 상단의 PITCH 상수(center/width/length/yaw)를 보정하세요.
